## `sht-openapi-web-sdk-generator local`

Generate the code from a local OpenAPI V3 file.

```
USAGE
  $ sht-openapi-web-sdk-generator local --config <value>

FLAGS
  --config=<value>  (required) OpenAPI V3 configuration file.

DESCRIPTION
  Generate the code from a local OpenAPI V3 file.
```
