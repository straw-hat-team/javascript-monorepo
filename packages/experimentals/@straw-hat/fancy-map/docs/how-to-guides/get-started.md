# Get Started

1. Install the dependency:

   ```bash
   yarn add @straw-hat/fancy-map
   ```

2. Import the package:

   ```typescript
   import { FancyMap } from '@straw-hat/fancy-map';
   ```

3. Create a new map using the fancy map:

   ```typescript
   import { FancyMap } from '@straw-hat/fancy-map';
   const mymap = new FancyMap();
   ```
